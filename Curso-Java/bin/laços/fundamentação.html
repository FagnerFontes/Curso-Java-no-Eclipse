Estrutura de Repetição


    A estrutura de repetição (ou loop) em Java é usada para executar um bloco de código múltiplas vezes com base em uma condição. Existem três principais estruturas de repetição em Java: while, do-while e for. Cada uma tem suas próprias características e é adequada para diferentes tipos de situações. Vamos discutir cada uma delas com exemplos e fundamentos.

    • While
